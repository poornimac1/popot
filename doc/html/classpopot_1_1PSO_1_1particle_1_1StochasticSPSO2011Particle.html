Detailed Description

The template parameters is the problem you want to solve, and the parameters (inertia, accelaration) of the velocity update rule Compared to the base type BaseParticle, this type adds the some specific position and velocity update rules These come from the Standard PSO 2011 . The stochastic code simply reevaluates the fitness of the personal best position before possibly changing it.
