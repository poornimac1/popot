Detailed Description

template<typename PARAMS, typename PARTICLE, typename TOPOLOGY, typename STOP_CRITERIA>
class popot::PSO::algorithm::Base< PARAMS, PARTICLE, TOPOLOGY, STOP_CRITERIA >

Standard particle swarm optimization

Parameters:
PARAMSDefines the parameters of the swarm (e.g. the evaluation mode : synchronous or asynchronous)
PARTICLEThe particle type you want to use, e.g. swarm::particle::TraditionalParticle
TOPOLOGYDefines the topology, e.g. swarm::topology::VonNeuman
STOP_CRITERIADefines the condition to stop the evolution of the swarm
Examples:
example-001.cc, example-002.cc, and example-003.cc.
