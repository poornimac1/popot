70  /*
71 
72  KISS
73  http://www.helsbreth.org/random/rng_kiss.html
74 
75  the idea is to use simple, fast, individually promising
76  generators to get a composite that will be fast, easy to code
77  have a very long period and pass all the tests put to it.
78  The three components of KISS are
79  x(n)=a*x(n-1)+1 mod 2^32
80  y(n)=y(n-1)(I+L^13)(I+R^17)(I+L^5),
81  z(n)=2*z(n-1)+z(n-2) +carry mod 2^32
82  The y's are a shift register sequence on 32bit binary vectors
83  period 2^32-1;
84  The z's are a simple multiply-with-carry sequence with period
85  2^63+2^32-1. The period of KISS is thus
86  2^32*(2^32-1)*(2^63+2^32-1) > 2^127
87  */
