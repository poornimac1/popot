00071       KISS
00072       http://www.helsbreth.org/random/rng_kiss.html
00073 
00074       the idea is to use simple, fast, individually promising
00075       generators to get a composite that will be fast, easy to code
00076       have a very long period and pass all the tests put to it.
00077       The three components of KISS are
00078       x(n)=a*x(n-1)+1 mod 2^32
00079       y(n)=y(n-1)(I+L^13)(I+R^17)(I+L^5),
00080       z(n)=2*z(n-1)+z(n-2) +carry mod 2^32
00081       The y's are a shift register sequence on 32bit binary vectors
00082       period 2^32-1;
00083       The z's are a simple multiply-with-carry sequence with period
00084       2^63+2^32-1.  The period of KISS is thus
00085       2^32*(2^32-1)*(2^63+2^32-1) > 2^127
00086     */
